# 05. 戦闘システム仕様

## 戦闘フロー

```
攻撃側の攻撃 → [防御側が生存 & 射程内] → 反撃
             → [攻撃側が追撃条件を満たす] → 攻撃側の追撃
             → [防御側が追撃条件を満たす & 射程内] → 防御側の追撃
```

1. **攻撃側の攻撃**: 命中判定 → ダメージ適用
2. **反撃**: 防御側が生存 & 武器射程に攻撃側がいれば反撃
3. **追撃（攻撃側）**: 両者生存 & 攻撃側の攻速差 ≥ 5
4. **追撃（防御側）**: 両者生存 & 防御側の攻速差 ≥ 5 & 射程内

## ダメージ計算

`DamageCalc.calculateForecast()` で計算。

### 物理ダメージ

```
ダメージ = STR + 武器威力 + 三すくみ補正 - DEF - 地形防御補正
         （最低 0）
```

### 魔法ダメージ

```
ダメージ = MAG + 魔法威力 + 三すくみ補正 - RES - 地形防御補正
         （最低 0）
```

> 武器タイプが `MAGIC` の場合に魔法ダメージ計算を使用。

## 命中率計算

```
基本命中 = 武器命中 + SKL × 2 + LCK ÷ 2 + 攻撃側地形命中補正
回避     = SPD × 2 + LCK ÷ 2 + 防御側地形回避補正
命中率   = 基本命中 - 回避 + 三すくみ命中補正
         （0〜100 にクランプ）
```

> 攻撃側の地形命中補正（hitBonus）は攻撃側が立っている地形に基づく。森+10/山+15/砦+10/村+5。水域上のユニットは-15のペナルティ。

## 必殺率計算

```
基本必殺 = 武器必殺 + SKL ÷ 2
必殺回避 = LCK ÷ 2
必殺率   = 基本必殺 - 必殺回避
         （0〜100 にクランプ）
```

## 必殺ダメージ

```
必殺ダメージ = 通常ダメージ × 3
```

`GameConfig.CRITICAL_MULTIPLIER = 3`

## 追撃条件

```
攻速 = SPD - 武器重さ
追撃可能 = 自分の攻速 - 相手の攻速 ≥ 5
```

`GameConfig.DOUBLE_ATTACK_SPEED_DIFF = 5`

## 反撃条件

```
距離 = 攻撃側と防御側のマンハッタン距離
反撃可能 = 距離が防御側の武器射程に含まれる
```

例: 弓（射程2）は隣接する攻撃者に反撃できない。

## 武器三すくみ補正

| 関係 | ダメージ | 命中 |
|------|---------|------|
| 有利 | +1 (`WEAPON_TRIANGLE_DAMAGE_BONUS`) | +15 (`WEAPON_TRIANGLE_HIT_BONUS`) |
| 不利 | -1 | -15 |
| 中立 | 0 | 0 |

## 地形補正

| 地形 | 命中補正（攻撃側） | 回避補正（防御側） | 防御補正（防御側） |
|------|-------------|-------------|-------------|
| 平地 | 0 | 0 | 0 |
| 森 | +10 | +20 | +1 |
| 山 | +15 | +30 | +2 |
| 砦 | +10 | +20 | +3 |
| 水域 | -15 | -15 | 0 |
| 村 | +10 | +10 | +1 |

> 命中補正は攻撃側の地形に、回避補正・防御補正は防御側の地形に基づく。水域は飛行ユニットのみ停止可能で、命中・回避ともにペナルティがかかる。

## 経験値計算

```
獲得EXP = baseExp(30) + (相手Lv - 自分Lv) × 3 + 撃破ボーナス(20)
        （1〜100 にクランプ）
```

## 武器耐久消費

- 命中した場合のみ攻撃側の武器耐久を 1 消費
- 耐久が 0 になると武器使用不可

## BattleResult

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `attacker` | GameUnit | 攻撃ユニット |
| `defender` | GameUnit | 防御ユニット |
| `attacks` | List\<AttackResult\> | 各攻撃の結果リスト |
| `attackerDefeated` | Boolean | 攻撃側が戦闘不能か |
| `defenderDefeated` | Boolean | 防御側が戦闘不能か |
| `expGained` | Int | 獲得経験値 |

## AttackResult

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `attackerIsInitiator` | Boolean | 攻撃を仕掛けた側か |
| `damage` | Int | 実際のダメージ |
| `hit` | Boolean | 命中したか |
| `critical` | Boolean | 必殺が出たか |
